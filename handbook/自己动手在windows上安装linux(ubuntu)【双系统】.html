    自己动手安装linux(双系统)

    KlayPeter 2024/10/12

    # 准备#

    # 1.下载ubuntu系统镜像#

    Download Ubuntu Desktop | Download | Ubuntu (opens new window)

    # 2.准备一个u盘作为启动盘（需要格式化，最好空盘），这里需要一个工具帮助实现#

    USBWriter download | SourceForge.net (opens new window)

    # 3.电脑硬盘分盘（分盘详细的就不说了，选择100G左右，至少30G）#

    # 我装了一个硬盘，分了200G#

    # 正式安装#

    # 1.启动菜单，看自己电脑的U盘启动的是什么键#

    插入烧录好的U盘，电脑重启的时候不停的按F12 ，选择启动项的时候 选带有USB字样的

    # 2.点击进入，按图片顺序来一步一步完成#

    # 3.这个要切换启动系统的顺序，我的是按f2然后进入这个界面，大家可以查阅自己电脑的#

    # 然后重启#

    就会发现进入系统要进行选择了系统，那就赶紧进入linux享受吧！

    # 首次进入linux后 wifi网络显示没有，连wifi图标都没有，我慌了，后面上网查方法#

    # 方法:更新linux内核#

    回到windows

    搜索www.bing,com

    将这四个文件下载, 用copy到U盘然后再迁移重启就可以了
